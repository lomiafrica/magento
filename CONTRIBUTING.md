# Contributing to lomi. Magento 2 payments

Thanks for helping improve the lomi. Magento integration.

## Issues

- Use clear steps to reproduce, Magento version, PHP version, and module version.
- Include relevant logs (with secrets redacted).

## Pull requests

- Follow existing code style and Magento conventions.
- Test checkout redirect, callback return, and webhook handling when you touch payment flow.
- Do not commit real API keys or webhook secrets.
